# PocketVoice Backend

Node.js + MongoDB backend for the PocketVoice iOS app.

## Quick start

```bash
npm install
cp .env.example .env    # fill in values
npm run dev
```

Server listens on `http://localhost:3000`.

## Scripts

- `npm run dev` — watch mode (Node `--watch`)
- `npm start` — production mode
- `npm test` — run tests once
- `npm run test:watch` — tests in watch mode

See `docs/superpowers/specs/2026-04-22-pocketvoice-backend-design.md` for the full spec.
